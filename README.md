# Akizuki Forest

First console game of Ivy Cafeteria.

## Basic Usage

> [!WARNING]
> Your terminal must support Japanese output.

### Windows

```ps1
.\AkizukiForest.exe
```

### Linux

```sh
# sudo chmod a+x ./AkizukiForest
./AkizukiForest
```
